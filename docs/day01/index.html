  • Processing Hi-C data from the start [60min] [HTML | Rmd ]

    You will be streamlining the process from the previous exercise to make it more reproducible, easily tweak parameters or add new samples. We will be adding another yeast library in M phase (SRX5559676 ) and doing basic comparisons between the conditions.
